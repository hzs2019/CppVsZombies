项目简介

C++ vs. Zombies 是一个以 C++ 语言编写的用于植物大战僵尸的键控框架， 如果读者以前学习或使用过 pvz.py，那么你将会更快的掌握 pvz.h。 但是使用 pvz.h 的前提是对 C/C++ 基础语法有一定的了解， 若读者以前没有学习过 C/C++，你也许能够看懂示例脚本，但是当自己编写脚本时会遇到非常多的语法错误， 因此希望读者在自己编写脚本前学习一些相关的语法知识。
注意：该项目仅支持英文原版，不支持任何其他版本
如果在使用过程中遇到问题或者感觉函数使用不方便请联系作者，QQ：2360900496，昵称：向量

语言基础

对于 pvz.h 来说，学习 C语言 还是学习 C++ 没有太大的区别， 所以你可以随便选一个学习，但是不论选择那个，你必须看的语法知识有：
基本语法 数据类型（你可以将 float double 理解为小数类型）
运算符 判断 循环 函数 注释
推荐一个 C++ 教程网站： C++ 菜鸟教程
如果感觉看的很吃力那么你可以看 C语言 教程网站： C语言 菜鸟教程
